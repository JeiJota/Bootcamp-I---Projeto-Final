PortfolioHUB

Documentação de Implantação e Integração com GitHub

1. Planejamento da Implantação

O PortfolioHUB é uma plataforma web centralizada para exibição e gerenciamento de projetos e portfólios digitais.

  • Site de portfólio pessoal (frontend estático).
  • Repositório centralizado no GitHub para versionamento.
  • Integração em tempo real com a API do GitHub.
  • Políticas de segurança e controle de acesso configuradas.

2. Configuração Inicial e Integração

  • Repositório Bootcamp-Final criado no GitHub.
  • Branch padrão: main com GitHub Pages ativo.
  • Estrutura: index.html, style.css, script.js, SECURITY.md, README.md.
  • Integração com GitHub REST API v3 para dados dinâmicos.

3. Gestão de Usuários e Segurança

  • Autenticação 2FA ativa.
  • Branch Protection Rules configuradas.
  • Dependabot, Secret Scanning e CodeQL habilitados.
  • Arquivo SECURITY.md documentando vulnerabilidades.

4. Compartilhamento e Controle de Acesso

  • Modelo trunk-based development com main protegida.
  • Commits semânticos seguindo Conventional Commits.
  • Deploy automático via GitHub Pages.

5. Testes e Validação

  • Funcionalidades testadas: modais, navegação, integração API.
  • Bugs corrigidos: cards de projetos, navegação por teclado.
  • Validação final com Google Gemini confirmando boas práticas.

6. Revisão Final e Apresentação

  • Resumo das etapas concluídas com sucesso.
  • Desafios superados: limites da API, segurança em repositório público.
  • Apresentação em vídeo disponível no YouTube.
